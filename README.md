# Initialzing Gof Coach 

## method 1: Download Program Folder

step1.  Download the Code from github

step2.  Unzip the Downloaded zipped file on the Desktop window

step3.  oepn vscode for the folder on the desktop

in the Vscode

step4.  open a new powershell terminal

step5.  create a virtual environment with "python -m venv .venv"

step6.  activate .venv with ".venv/Scripts/activate"

step7.  install dependencies with "pip install -r requirements.txt"

step8.  run the app.py with "streamlit run app.py"

## method 2: Clone the repository

From Window terminal

step1. Click Right Button => 터미널에서 열기로 Power Shell Terminal Open

step2. Power Shell Terminal 에서 Clone the repository "clone https://github.com/leekwansoo/golf-coach.git"

a new folder will be downloaded on the desktop window terminal

from step3. to step8 : method 1 과 같이 실행
